# 讽刺营销版设计文档

[English Version](../en/marketing-satire.md)

## 项目目的

通过创建一个与科学体验版**技术完全相同**、仅 UI 文案和视觉风格不同的"过度营销版"，让用户通过一键切换直观感受：同一段音频，在不同包装下给人的心理暗示有多巨大。

**核心讽刺点：你花钱买的"高级功能"，可能只是文案换了一层皮。**

## 技术实现

- **音频处理代码零修改** — 底层 AudioEngine、AudioWorklet、Web Worker 完全不变
- CSS 主题通过 `body.marketing` 类切换
- 营销文案作为额外的 i18n 语言包加载（`zh-marketing.js` / `en-marketing.js`）
- 版本状态存储在 `localStorage('bb-edition')`
- URL 参数 `?edition=marketing` 支持直接分享营销版链接

## 讽刺效果预期

用户切换时会注意到：
1. **声音完全一样** — 没有任何音频变化
2. **参数完全一样** — 只是标签变了
3. **信息面板天差地别** — 物理事实 vs 玄学话术
4. **为什么我在营销版似乎"真的觉得放松了"？** — 这就是期望效应
