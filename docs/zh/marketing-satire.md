# 讽刺版设计文档（营销版 + 神棍版）

[English Version](../en/marketing-satire.md)

## 项目目的

通过创建一个与科学体验版**技术完全相同**、仅 UI 文案和视觉风格不同的"过度营销版"，让用户通过一键切换直观感受：同一段音频，在不同包装下给人的心理暗示有多巨大。

**核心讽刺点：你花钱买的"高级功能"，可能只是文案换了一层皮。**

## 版本切换

三个版本之间可任意切换，每个版本底部有两个按钮指向另外两个版本：

| 版本 | 按钮 1 | 按钮 2 |
|------|--------|--------|
| 科学版 | 🛒 看看智商税版本 → 营销版 | 🔮 看看神棍版本 → 神棍版 |
| 营销版 | 🔮 不够？试试开光加持版 → 神棍版 | 📉 降级到没有灵魂的版本 → 科学版 |
| 神棍版 | 🛒 降级到没开光的智商税版 → 营销版 | 📉 降级到没有灵魂的版本 → 科学版 |

URL 参数直达链接：
- 科学版：`?edition=science`
- 营销版：`?edition=marketing`
- 神棍版：`?edition=mystic`

通过 `?edition` 参数直接访问时，切换按钮会自动隐藏，让访问者先专注体验当前版本。

## 技术实现

- **音频处理代码零修改** — 底层 AudioEngine、AudioWorklet、Web Worker 完全不变
- CSS 主题通过 `body.marketing` / `body.mystic` 类切换
- 文案作为额外的 i18n 语言包加载（`zh-marketing.js` / `en-marketing.js` / `zh-mystic.js` / `en-mystic.js`）
- 版本状态存储在 `localStorage('bb-edition')`
- URL 参数 `?edition=marketing` 或 `?edition=mystic` 支持直接分享特定版本链接
- 通过 URL 参数访问时自动隐藏版本切换按钮，让访问者先专注体验

## 讽刺效果预期

用户切换时会注意到：
1. **声音完全一样** — 没有任何音频变化
2. **参数完全一样** — 只是标签变了
3. **信息面板天差地别** — 物理事实 vs 伪科学话术 vs 灵修神棍话术
4. **三个版本形成递进讽刺** — 科学版说真话，营销版吹牛，神棍版将牛吹上天
5. **为什么我在神棍版似乎“真的觉得灵魂被洗涤了”？** — 这就是期望效应
